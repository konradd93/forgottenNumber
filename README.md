# Forgotten Number

## Input
- int[] - array of integers
- range of integers: (-100000,100000)

## Output
- int - first forgotten integer greater than zero


## Example
### 1
input: {-2,0,1}
expected: 2
### 2
input: {-2,0,2}
expected: 1
### 3
input: {-2,-1,-5}
expected: 1
### 4
input: {}
expected: 1
